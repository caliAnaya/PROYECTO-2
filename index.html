CSS

En esta pagina se hace una breve descripción de los antecedentes, CSS fue desarrollado por W3C (World Wide Web Consortium) en 1996 por una razón muy sencilla. CSS permite estilizar todo en un archivo diferente, creando el estilo allí y después integrando el archivo CSS sobre el marcado HTML. Esto hace que el marcado HTML sea mucho más limpio y fácil de mantener. En resumen, con CSS no tienes que describir repetidamente cómo se ven los elementos individuales. Esto ahorra tiempo, hace el código más corto y menos propenso a errores.
